Hammer Game

Developed an interactive digital hammer game inspired by traditional carnival strength testers, using a Raspberry Pi and Python. The game featured a beam break sensor to detect player input (simulating a hammer strike) and a vertical LED strip to visually represent the force of the hit. Designed custom LED animations for different game states, including idle mode, active strike visualization, and victory or failure feedback. Programmed responsive game logic in Python to calculate and display results in real time, ensuring smooth transitions and engaging visual effects. Consistently used at UCLA hill events.

Line-Following Robot Car

Designed and programmed an autonomous line-tracking robot car using Arduino to navigate a course with complex challenges, including disconnected paths and decoy black bars. The robot used an array of infrared sensors to detect line position, and implemented a custom PID control algorithm to adjust motor speed and steering for smooth, accurate line following. The PID controller was tuned to minimize oscillations and overshooting, enabling the robot to respond precisely to sharp turns, gaps, and false signals. The system demonstrated robust performance and adaptability in a dynamic test environment.

Carti: 3-Stage Robot Arm

Developed control software for a 3-stage robotic arm used in an FRC (FIRST Robotics Competition) robot that competed at the 2023 World Championship in Houston. The arm's motion was governed by inverse kinematics and coordinated multi-joint control, enabling smooth and accurate positioning. Implemented and fine-tuned custom PID control loops for each joint, accounting for joint coupling and dynamic load variations. Emphasized real-time responsiveness and stability under high-speed, high-stress conditions. This system enabled the robot to reliably manipulate game pieces with precision during autonomous and teleoperated periods, contributing to competitive performance at the highest level of play.

Bilbert: Vision-Guided Flywheel Shooter

Programmed a precision flywheel shooter system for an FRC robot that reached the finals at Beach Blitz 2022. The mechanism included a 360-degree rotating turret, an adjustable hood for variable shot trajectories, and a dual-wheel flywheel for consistent ball velocity. Integrated a Limelight vision system to enable real-time target tracking and distance estimation. Developed C++ control algorithms to calculate optimal flywheel speed and hood angle based on target position. Tuned multiple PID controllers to manage turret rotation, hood positioning, and flywheel velocity, ensuring rapid and accurate adjustments during autonomous and teleoperated periods. The system performed reliably under high-pressure match conditions, significantly improving scoring consistency.

Light Tracking Solar Panel

Designed and built a light-tracking solar panel system that dynamically adjusts its orientation to maximize sunlight intake. The setup used two light-dependent resistors (LDRs) positioned on either side of the panel to detect the direction of the strongest light source. An Arduino microcontroller read the LDR values and controlled a servo motor to rotate the panel toward the brighter side. Implemented analog signal processing and threshold logic in Arduino C/C++ to ensure responsive, stable tracking behavior. A USB port was custom-soldered to the panel to serve as a power output, allowing the solar panel to function as a mobile energy source for low-power devices.
